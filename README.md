# 👽 Showcase of various DevExtreme components 👽

## 🚀 Getting Started 🚀

### Clone the repository

```bash
git clone https://github.com/ptaushanov/devextreme-showcase.git
cd devextreme-showcase
```

### Install dependencies

```bash
npm install
```

## ⚙️ Running the project ⚙️

```bash
npm run dev
```
